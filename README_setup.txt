1. Install Python 3.10.11
2. Install XAMPP
3. Start Apache and MySQL
4. Open phpMyAdmin
5. Create database: szic_attendance
6. Import: szic_attendance_final.sql
7. Open project folder in VS Code
8. Create venv:
   py -3.10 -m venv fyp_env
9. Activate:
   .\fyp_env\Scripts\Activate.ps1
10. Install:
   pip install -r requirements.txt
11. Run:
   python app.py
12. Open:
   http://127.0.0.1:5000
